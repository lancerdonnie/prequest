# Prequest

A REST request tool built with svelte. Inspired by postman

## Tech Stack

Svelte, Codemirror, Typescript

## Running

Run this project

```bash
  git clone https://github.com/lancerdonnie/prequest
  cd prequest
  npm install
  npm run dev
```
